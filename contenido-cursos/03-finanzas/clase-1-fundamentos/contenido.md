## Contenido

### 1. ¿Qué es la Finanza?

La finanza es la ciencia que estudia la administración del dinero y los recursos económicos de individuos, empresas y gobiernos.

**Áreas principales:**
- **Finanzas Personales**: Gestión de dinero individual
- **Finanzas Corporativas**: Gestión de empresas
- **Finanzas Públicas**: Gestión de recursos del estado
- **Finanzas Internacionales**: Transacciones entre países

### 2. Conceptos Básicos

#### Activo
Todo lo que tiene valor y pertenece a una persona o empresa.

**Tipos:**
- **Activo Circulante**: Efectivo, cuentas por cobrar, inventario
- **Activo Fijo**: Inmuebles, equipos, vehículos

```
Activos = Pasivos + Patrimonio
```

#### Pasivo
Deudas u obligaciones de una persona o empresa.

**Tipos:**
- **Pasivo Circulante**: Cuentas por pagar, préstamos corto plazo
- **Pasivo Fijo**: Préstamos largo plazo, hipotecas

#### Patrimonio
Lo que queda después de restar pasivos de activos.

```
Patrimonio = Activos - Pasivos
```

### 4. Indicadores Financieros

#### Razones de Liquidez
Capacidad de pagar deudas corto plazo.

```
Liquidez Corriente = Activos Circulantes / Pasivos Circulantes
                   = $100,000 / $25,000
                   = 4.0

Interpretación: Por cada $1 de deuda, tiene $4 para pagar
```

#### Razones de Rentabilidad
Qué tan bien genera ganancias.

```
Margen de Ganancia = Utilidad Neta / Ventas
                   = $80,500 / $500,000
                   = 16.1%

ROE = Utilidad Neta / Patrimonio
    = $80,500 / $205,000
    = 39.3%
```

#### Razones de Endeudamiento
Proporción de deuda vs patrimonio.

```
Razón de Endeudamiento = Pasivo Total / Patrimonio
                       = $125,000 / $205,000
                       = 0.61

Interpretación: Tiene $0.61 de deuda por cada $1 de patrimonio
```
