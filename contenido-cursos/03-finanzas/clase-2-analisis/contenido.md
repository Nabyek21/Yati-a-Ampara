### 1. Análisis de Mercado (FODA)

#### Fortalezas
Aspectos internos positivos de la empresa.

```
Ejemplos:
- Marca establecida
- Tecnología superior
- Equipo experimentado
- Eficiencia operacional
- Productos de calidad
```

#### Oportunidades
Factores externos que podrían favorecer el crecimiento.

```
Ejemplos:
- Mercados emergentes
- Nuevas regulaciones favorables
- Cambios demográficos
- Tendencias tecnológicas
- Alianzas estratégicas
```

#### Debilidades
Aspectos internos que necesitan mejora.

```
Ejemplos:
- Recursos limitados
- Marca débil
- Procesos ineficientes
- Falta de expertise
- Dependencia de terceros
```

#### Amenazas
Factores externos que podrían perjudicar el negocio.

```
Ejemplos:
- Competencia fuerte
- Cambios regulatorios
- Crisis económica
- Cambios en preferencias
- Tecnología disruptiva
```

### 2. Métodos de Evaluación de Inversiones

#### Valor Presente Neto (VPN)

Calcula el valor actual de los flujos futuros.

```
VPN = -Inversión Inicial + ∑(Flujo_t / (1 + i)^t)

Donde:
- Flujo_t = Flujo en el año t
- i = Tasa de descuento
- t = Número de períodos

Criterio:
- VPN > 0: Aceptar proyecto
- VPN < 0: Rechazar proyecto
- VPN = 0: Indiferente
```

**Ejemplo:**
```
Inversión: $100,000
Tasa: 10%

Año 1: $30,000 → VPN = $30,000 / 1.10 = $27,272.73
Año 2: $35,000 → VPN = $35,000 / 1.21 = $28,925.62
Año 3: $40,000 → VPN = $40,000 / 1.331 = $30,026.10

VPN Total = -$100,000 + $27,272.73 + $28,925.62 + $30,026.10
          = -$13,775.55 (RECHAZAR)
```

#### Tasa Interna de Retorno (TIR)

La tasa de descuento que iguala ingresos y egresos.

```
Criterio:
- TIR > Tasa Mínima: Aceptar
- TIR < Tasa Mínima: Rechazar

Ejemplo:
Tasa Mínima = 15%
TIR Proyecto = 18%
Decisión: ACEPTAR (18% > 15%)
```

#### Período de Recuperación

Tiempo necesario para recuperar la inversión.

```
Período = Año en que se recupera inversión

Ejemplo:
Año 0: -$100,000
Año 1: +$30,000 (Acumulado: -$70,000)
Año 2: +$35,000 (Acumulado: -$35,000)
Año 3: +$40,000 (Acumulado: +$5,000)

Período de Recuperación = 2 años + ($35,000/$40,000) = 2.875 años
```

#### Índice de Rentabilidad

```
Índice = VPN Positivo / Inversión Inicial

Criterio:
- Índice > 1: Aceptar
- Índice < 1: Rechazar

Ejemplo:
VPN = $50,000 / Inversión = $100,000
Índice = $50,000 / $100,000 = 0.50 (RECHAZAR)
```

### 3. Tipos de Análisis

#### Análisis Fundamental
Evalúa el valor real basado en fundamentales.

```
Factores a considerar:
- Ganancias históricas
- Crecimiento esperado
- Márgenes de ganancia
- Posición competitiva
- Calidad de gestión
```

#### Análisis Técnico
Estudia precios históricos para predecir tendencias.

```
Herramientas:
- Gráficos de precios
- Soportes y resistencias
- Promedios móviles
- Indicadores (RSI, MACD)
```

### 4. Riesgo y Retorno

#### Relación Riesgo-Retorno

```
      Retorno (%)
         |
      30 |        Acciones
         |       /
      20 |      /
         |     / Bonos
      10 |    /
         |   / Depósitos
       0 |__/
         └─────────────────── Riesgo
           0    5    10   15   20
```

**Mayor riesgo = Mayor retorno esperado**

#### Diversificación

```
Riesgo del Portafolio

Sin Diversificación          Con Diversificación
Riesgo Alto                  Riesgo Bajo

100% Acciones X              40% Acciones
                             30% Bonos
                             30% Efectivo

Volatilidad: 25%             Volatilidad: 8%
```

### 5. Indicadores de Mercado

#### Razón P/E (Precio-Ganancia)
```
P/E = Precio Acción / Ganancia por Acción

Interpretación:
- P/E bajo: Puede estar subvaluado
- P/E alto: Puede estar sobrevaluado
- Comparar con competidores
```

#### Capitalización de Mercado
```
Market Cap = Precio Acción × Número de Acciones

Ejemplo:
Precio: $50 por acción
Acciones: 1,000,000
Market Cap = $50,000,000 (empresa de $50M)
```

#### Dividend Yield
```
Yield = Dividendo por Acción / Precio Acción

Ejemplo:
Dividendo: $2 por acción
Precio: $50
Yield = $2/$50 = 4%
```
