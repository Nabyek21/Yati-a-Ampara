## Tema 1: Introducción al Álgebra

### ¿Qué es el álgebra?

El álgebra es una rama de las matemáticas que utiliza símbolos (generalmente letras) para representar números desconocidos o valores generales. Estos símbolos, llamados variables, nos permiten escribir ecuaciones y resolver problemas de forma generalizada.

**Ejemplo:**
- En lugar de decir "un número más 5 es igual a 12"
- Escribimos: x + 5 = 12
- Donde x es el número desconocido

### Conceptos básicos

#### Variables
- Son símbolos (generalmente letras) que representan valores desconocidos
- Ejemplos: x, y, a, b, z
- Pueden tomar cualquier valor numérico

#### Constantes
- Son valores numéricos fijos
- Ejemplos: 3, -5, 0.75, π

#### Operadores
- Suma: +
- Resta: -
- Multiplicación: ×, ·, o simplemente xY (cuando x e Y son variables)
- División: ÷, /
- Potencia: ^, o superíndice

#### Expresiones algebraicas
- Combinaciones de variables, constantes y operadores
- Ejemplos: 2x + 3, y² - 4y + 1, 5a + 2b - 7

### Ecuaciones algebraicas

Una ecuación es una igualdad entre dos expresiones algebraicas. El objetivo es encontrar el valor de la variable que hace que la ecuación sea verdadera.

**Estructura básica:**
```
Lado izquierdo = Lado derecho
   2x + 3   =      11
```

**Tipos de ecuaciones:**
1. Ecuaciones lineales (grado 1): 2x + 3 = 11
2. Ecuaciones cuadráticas (grado 2): x² - 4x + 3 = 0
3. Ecuaciones cúbicas (grado 3): x³ - 2x = 0

## Tema 2: Ecuaciones Lineales

### Definición

Una ecuación lineal es una ecuación de primer grado (la variable tiene exponente 1).

**Forma general:** ax + b = c

Donde:
- a, b, c son constantes
- x es la variable
- a ≠ 0

### Propiedades fundamentales

Para resolver una ecuación, podemos:
1. Sumar o restar el mismo número a ambos lados
2. Multiplicar o dividir ambos lados por el mismo número (excepto 0)

**Estas operaciones mantienen la igualdad.**

### Método de resolución

**Pasos:**
1. Identifica términos con la variable en un lado
2. Identifica términos constantes en el otro lado
3. Simplifica cada lado
4. Aísla la variable

**Ejemplo resuelto:**

```
Resolver: 2x + 3 = 11

Paso 1: Restar 3 de ambos lados
        2x + 3 - 3 = 11 - 3
        2x = 8

Paso 2: Dividir ambos lados entre 2
        2x ÷ 2 = 8 ÷ 2
        x = 4

Verificación: 2(4) + 3 = 8 + 3 = 11 ✓
```

### Ejemplos adicionales

**Ejemplo 2:**
```
3x - 5 = 10
3x - 5 + 5 = 10 + 5
3x = 15
x = 5
```

**Ejemplo 3:**
```
-2x + 8 = 2
-2x + 8 - 8 = 2 - 8
-2x = -6
x = 3
```

**Ejemplo 4 (con fracciones):**
```
(x/2) + 3 = 7
(x/2) + 3 - 3 = 7 - 3
x/2 = 4
x = 8
```

## Tema 3: Sistemas de Ecuaciones

### Definición

Un sistema de ecuaciones es un conjunto de dos o más ecuaciones que deben resolverse simultáneamente.

**Ejemplo:**
```
2x + y = 7
x - y = 2
```

### Métodos de resolución

#### Método 1: Sustitución

1. Despeja una variable en una ecuación
2. Sustituye en la otra ecuación
3. Resuelve
4. Sustituye el valor encontrado para hallar la otra variable

**Ejemplo:**
```
Ecuación 1: 2x + y = 7
Ecuación 2: x - y = 2

De la ecuación 2: x = y + 2

Sustituyendo en ecuación 1:
2(y + 2) + y = 7
2y + 4 + y = 7
3y + 4 = 7
3y = 3
y = 1

Entonces: x = 1 + 2 = 3

Solución: x = 3, y = 1
```

#### Método 2: Eliminación

1. Multiplica las ecuaciones para hacer coeficientes iguales
2. Suma o resta para eliminar una variable
3. Resuelve para la variable restante
4. Sustituye para encontrar la otra

**Ejemplo:**
```
Ecuación 1: 2x + y = 7
Ecuación 2: x - y = 2

Sumando las ecuaciones:
2x + y + (x - y) = 7 + 2
3x = 9
x = 3

Sustituyendo en ecuación 2:
3 - y = 2
y = 1

Solución: x = 3, y = 1
```
